# 猛蹬小店功能说明

## 当前功能

猛蹬小店用于自动完成 GPT 账号邮箱注册，并在授权完成后生成内置 Codex JSON。系统入口为浏览器扩展侧边栏，用户配置运行次数、邮箱服务、接码服务和必要账号信息后，可单步执行或自动执行完整流程。侧边栏的邮箱注册信息区域提供网页 access token 注册开关，用户可选择只完成网页账号注册并导出 ChatGPT 网页 access token。

## 来源与输出

来源固定为 `内置 Codex JSON`。网页 access token 注册开关关闭时，授权成功后扩展根据 OAuth 回调信息生成 Codex 认证 JSON，并通过浏览器下载能力保存到本地下载目录 `Codex-OAuth-JSON`。

网页 access token 注册开关开启时，当前流程只执行到 Step 6。系统在网页账号注册完成并等待页面稳定后，进入 ChatGPT 网页会话读取 access token，并将当前批次采集到的 token 下载到本地下载目录 `ChatGPT-Web-Access-Tokens`。自动运行同一批次使用同一个 txt 文件，文件内容为一行一个 access token；单次运行也按同样格式输出。该模式不继续执行 OAuth 登录、OAuth 确认和平台回调验证步骤，也不生成 Codex JSON。

## 邮箱流程

注册方式固定为邮箱注册。注册邮箱由 DuckDuckGo Email Protection 生成，用户可点击侧边栏中的获取按钮生成 `@duck.com` 邮箱，也可手动填写邮箱。

邮箱服务支持 QQ 邮箱、2925 邮箱与验证码平台。QQ 邮箱用于轮询验证码邮件；2925 邮箱作为收信服务时使用接收邮箱模式，并可通过 2925 账号池维护登录账号；验证码平台通过 `https://code.youkeduo.site` 按当前注册邮箱直接查询线上验证码，不再打开网页邮箱页面。

## 接码流程

接码设置只用于邮箱注册后的手机号验证兜底，不提供手机号注册入口。接码服务仅支持 HeroSMS 与 SMSBower。系统可按用户配置的服务商顺序获取号码、轮询验证码并提交。

每次需要手机号验证时，系统会按当前接码服务重新获取号码，不会复用历史号码或手动记录号码。

## 界面与更新

侧边栏左上角品牌显示为猛蹬小店，点击不会跳转外部页面。邮箱注册信息区域包含 OAuth 超时、Cookies 清理、网页 access token 注册等运行开关；开启网页 access token 注册后，步骤列表仅显示 Step 1 到 Step 6，Step 6 显示为下载网页 AccessToken。扩展不包含新手引导、在线公告或在线版本检测。

## 影响范围

旧配置导入到简化版后，会被归一化为支持范围内的配置：来源归一为内置 Codex JSON，邮箱生成归一为 DuckDuckGo，邮箱服务归一为 QQ、2925 或验证码平台，接码服务归一为 HeroSMS 或 SMSBower。网页 access token 注册开关默认为关闭，关闭时保持完整 Codex JSON 流程。
